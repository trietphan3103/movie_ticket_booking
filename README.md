## 1) Run server:

-   Run command

```
    docker network create Banana_cinema
    docker-compose up -d
```

-   Your website has now running on port: 8080 (http://localhost:8080/)

-   Sql server's port: 3399:3306

-   Phpadmin's port: 8888 (http://localhost:8888/)
